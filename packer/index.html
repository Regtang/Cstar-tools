0. 产品定位

                
喜事达（Cstar）装箱软件面向业务、仓库和管理复核人员，提供离线装箱计划、数据导入、多箱分装、箱门校验、分层查看、装载顺序和报告导出。

              

              

                
1. 标准作业流程

                
    
                  
  1. 填写任务名称、客户、操作员和装箱日期。
    2. 
                  
  2. 优先使用专业模板录入货物，避免字段名和单位填错。
    3. 
                  
  3. 选择箱型、最大箱数和约束规则，点击自动装箱。
    4. 
                  
  4. 按箱号和分层高度检查视图、未装货物、重心、利用率和智能建议。
    5. 
                  
  5. 在装载视图下选择具体货物，用方向按钮进行 100 毫米微调。
    6. 
                  
  6. 导出装箱表格、复制报告，或打印存档。
    7. 
                

              

              

                
2. 导入文件规范

                
    
                  
  1. 工作簿：读取第一个工作表，第一行必须是字段名。
    2. 
                  
  2. 文字文档：读取文档中的货物表格，第一行必须是字段名。
    3. 
                  
  3. 逗号表：第一行字段名，后续行是货物数据。
    4. 
                  
  4. 推荐使用专业模板中的中文字段：名称、长、宽、高、重量、数量、分组、可旋转、可倾斜、可堆叠、优先级。
    5. 
                

              

              

                
3. 模板填写要求

                
尺寸单位为毫米，重量单位为千克。数量必须为正整数。可旋转、可倾斜、可堆叠使用下拉选择。优先级使用高、中、低。专业模板内置下拉、数据校验、字段字典、箱型参考和示例数据。

              

              

                
4. 多箱与箱门校验

                
最大箱数控制可使用的集装箱数量。软件会按箱号生成分装方案，并校验货物是否能通过箱门。若提示无法通过箱门，需要调整包装方向、拆分货物或更换运输方案。

              

              

                
5. 对标成熟软件新增能力

                
参考成熟装箱软件常见能力，当前版本增加分组/目的地、可倾斜限制、最大堆叠层数、装载步骤回放、箱底承重热区、重量分布风险提示、专业模板和正式用户手册。

              

              

                
6. 分层与移动

                
小包装较多时，拖动分层显示高度，只查看指定高度以下的货物。选择具体货物后，可使用方向按钮按 100 毫米微调。系统会阻止越界和碰撞。

              

              

                
7. 分享与交接

                
点击导出项目得到项目文件，适合同事继续编辑同一个任务。发给同事时建议同时发送软件文件夹、项目文件、专业模板和用户手册。

              

              

                
8. 现场复核

                
当前算法适合快速生成仓库作业参考方案。正式发运前仍需由现场人员复核承重、绑扎、禁配、法规和箱况。

              

              

                
9. 常见问题

                
导入后没有货物时，先检查第一行字段名。货物无法装入时，检查尺寸、重量、箱门、不可堆叠和最大箱数。同事看到旧版时，重新解压最新版或刷新浏览器缓存。
