# 喜事达（Cstar）装箱软件

这是一个可离线分享的集装箱装箱计划工具。把整个软件文件夹发给同事，对方直接用浏览器打开首页文件即可使用。

## 功能

- 20GP、40GP、40HQ、45HQ 箱型选择
- 货物清单录入、逗号表、工作簿、文字文档导入，支持专业模板下载
- Cstar 品牌界面和中文交付文件
- 分组/目的地、可倾斜限制和建议装载顺序
- 左右重量分布风险提示
- 自动装箱、重货优先、不可堆叠、左右重量均衡估算
- 重不压轻安全约束：较重货物不会被压在更轻货物之上
- 全局择优装箱：在所有可用空间中比选位置，减少碎片、提升空间利用率
- 多箱装载计划，支持设置最大箱数并按箱号查看结果
- 全新 3D 交互视图：拖动旋转、滚轮缩放、右键/双指平移、双击复位（纯离线）
- 等距、俯视、热力、侧视多种装载视图
- 箱型承重按「最大总重 − 箱体自重」计算净载重（已修正 40 尺箱承重高于 20 尺箱的错误）
- 箱门尺寸校验，防止货物无法通过箱门
- 分层查看、货物选择和 100 毫米步进微调
- 空间利用率、重量利用率、重心、未装货物提示和智能装载建议
- 本机自动保存、完整项目文件导入导出
- 报告复制、装箱结果表格导出、打印存档
- 正式用户手册：`docs/喜事达Cstar装箱软件用户手册.docx`
- 专业导入模板：`templates/Cstar货物导入模板.xlsx`

## 使用方式

1. 打开首页文件。
2. 填写任务名称、客户/订单、操作员和装箱日期。
3. 在“货物清单”页录入货物，或点击“下载模板”后使用专业模板导入。
4. 在“工作台”选择箱型、最大箱数和约束规则，点击“自动装箱”。
5. 按箱号和分层高度检查未装货物、重心、智能建议和视图结果。
6. 如需微调，选择具体货物后使用方向按钮移动，系统会阻止越界和碰撞。
7. 在“装箱报告”页复制报告、导出表格或打印存档。

## 分享项目

- 点击“导出项目”会生成完整项目文件。
- 同事收到本文件夹和项目文件后，打开首页文件，点击“打开项目”即可继续编辑。
- 浏览器会自动保存最近一次项目，但跨电脑协作应以导出的项目文件为准。

## 导入字段

推荐优先使用 `templates/Cstar货物导入模板.xlsx`。模板包含中文字段说明、下拉选项、数据校验、箱型参考和示例数据。

软件支持三种货物清单格式：

- 逗号表：第一行字段名，后续行为货物数据。
- 工作簿：读取第一个工作表，第一行字段名。
- 文字文档：读取文档中的货物表格，第一行字段名。

暂不支持老式二进制表格和文档。可以先另存为新版工作簿或文字文档。

| 字段 | 含义 |
| --- | --- |
| 名称 | 货物名称 |
| 长 | 长度，单位毫米 |
| 宽 | 宽度，单位毫米 |
| 高 | 高度，单位毫米 |
| 重量 | 单件重量，单位千克 |
| 数量 | 数量，正整数 |
| 可旋转 | 是否可旋转 |
| 可倾斜 | 是否可倾斜 |
| 可堆叠 | 是否可堆叠 |
| 优先级 | 3 高，2 中，1 低 |

## 注意

当前算法用于快速生成装柜作业参考方案。正式发运前，需要现场人员复核实际货物包装、承重、绑扎、禁配、法规、箱况和安全要求。
