# 喜事达AEO —— 从买服务器到打开登录页（Ubuntu + Docker 极简版）

面向非运维人员，照着复制粘贴即可。全程约 20 分钟。
适用：阿里云 / 腾讯云 **轻量应用服务器**，系统 **Ubuntu 22.04**。访问方式用 **IP + 端口**（无需域名、无需备案）。

---

## 第 1 步：买一台服务器

1. 打开 [阿里云轻量](https://www.aliyun.com) 或 [腾讯云轻量](https://cloud.tencent.com)，注册并完成**实名认证**。
2. 购买"轻量应用服务器"，选：
   - 地域：选离你近的（如华东-上海 / 华南-广州）
   - 镜像：**系统镜像 → Ubuntu 22.04**（若有"Docker"应用镜像可直接选，能省第 4 步）
   - 配置：2核2G 起步即可
3. 购买后在控制台**设置/重置实例密码**，记下：
   - 公网 IP（形如 `123.45.67.89`）
   - root 密码

---

## 第 2 步：放行 8000 端口（很重要，否则打不开）

在轻量服务器控制台 → **防火墙 / 安全组** → 添加规则：
- 协议 **TCP**，端口 **8000**，来源 `0.0.0.0/0`（或仅限你公司出口 IP 更安全）

---

## 第 3 步：从你的 Mac 连上服务器并上传文件

打开 Mac"终端"。

**① 上传项目**（把 `xishida-aeo` 文件夹传到服务器，注意把 IP 换成你的）：
```bash
cd ~/Desktop
scp -r xishida-aeo root@123.45.67.89:/opt/
```
首次会提示 yes，然后输入 root 密码。

**② 登录服务器：**
```bash
ssh root@123.45.67.89
```
（之后的命令都在服务器上执行）

---

## 第 4 步：安装 Docker（若买的是 Docker 镜像，跳过）

```bash
curl -fsSL https://get.docker.com | sh
```

---

## 第 5 步：配置并启动

```bash
cd /opt/xishida-aeo
cp deploy/.env.production .env
nano .env          # 改 AEO_ADMIN_PASSWORD 为你的密码，保存：Ctrl+O 回车，Ctrl+X 退出
docker compose up -d
```

等一两分钟（首次构建镜像）。检查：
```bash
docker compose ps                         # 状态 healthy
curl http://127.0.0.1:8000/api/health     # 返回 {"status":"ok"}
```

---

## 第 6 步：打开登录页

在任意浏览器访问：
```
http://你的公网IP:8000
```
用户名 `admin`，密码为你在 `.env` 里设置的那个。看到驾驶舱即上线成功 ✅

---

## 第 7 步：上线后必做（安全）

1. 「用户管理」给每位同事建独立账号、分配角色。
2. 「设置」里再改一次 admin 密码。
3. 删除/停用演示账号（guanwu / neishen / caiwu / wuliu / anbao）。
4. 「设置 → 清空业务数据」，清掉示例数据，开始录入真实数据。

---

## 日常维护

```bash
cd /opt/xishida-aeo
docker compose logs -f          # 看日志
docker compose restart          # 重启
docker compose up -d --build    # 更新代码后重建
```

## 备份（合规要求留存≥3年）
```bash
docker run --rm -v xishida-aeo_aeo-data:/data -v $(pwd):/backup alpine \
  tar czf /backup/aeo-backup-$(date +%F).tar.gz -C /data .
```
把生成的 `aeo-backup-*.tar.gz` 下载到本地或公司存储保存。

---

## 卡住了？

- **打不开页面**：检查第 2 步 8000 端口是否放行；`docker compose ps` 看是否 healthy。
- **scp/ssh 连不上**：确认 IP、root 密码正确，服务器已开机。
- **想加域名 + HTTPS**：见同目录 `生产上线手册.md` 第 3 节（需先给域名做 ICP 备案）。
